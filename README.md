# muon2023
These programs analyze cosmic muon data from the Muon Test Stand and Dark Box at the APD Lab. Each program was used for specific range of Runs.

- caen_muon_v01.ipynb
Run 25 (single run analysis)

- caen_muon_v01a.ipynb
Run26 (signle run analysis)

- caen_muon_v01b.ipynb
Run28 (single run analysis)

- caen_muon_v01c_multi_files.ipynb
Run41-49

- caen_muon_v01d_darkbox.ipynb
(dark box)Run63

- caen_muon_v01e_multi_files-TOA.ipynb
Run221-229
Run230-239
Run250-259
